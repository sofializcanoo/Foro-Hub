# Foro Hub - Challenge Alura

API REST desarrollada con **Java** y **Spring Boot** para gestionar tópicos de un foro, permitiendo operaciones CRUD (Crear, Leer, Actualizar, Borrar).

## 🛠️ Tecnologías utilizadas
* Java 17
* Spring Boot 3.x
* H2 Database (Memoria)
* Lombok
* Maven

## 🚀 Cómo probarlo
1. Clona este repositorio.
2. Ejecuta `mvn spring-boot:run` en la terminal.
3. Accede a `http://localhost:8080/topicos` para ver la lista de tópicos.
